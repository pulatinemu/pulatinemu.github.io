使用hexo搭建博客

John Doe2024-10-282024-10-28

下载软件

node.js 修改缓存位置

git Git安装教程

验证下载是否成功

下载hexo

在Github搭建仓库

仓库命名：用户名.github.io

public

添加readme文件

生成SHH Keys

任意文件夹内，右键，Git bash

一直回车

然后进入C:\Users\用户名，在里面进入.ssh文件

用记事本打开里面的id_rsa.pub,全选复制里面的代码

Github创建SHH

setting→SHH→新建→随意命名，默认类型→粘贴，创建

测试是否成功

git bash

显示成功连接

生成本地博客

依次输入

如果显示找不到命令，命令前添加npx

上传博客

进入Blog文件夹，用记事本打开_config.yml

拉到最下面将deploy后面的全删掉，复制粘贴这段

去github之前生成的仓库页面，点code，复制https链接

将其粘贴到我们记事本中的repository：后面 记得删除git clone

然后保存退出

回到博客文件夹，git bash

安装自动部署发布工具

第一次git，需要配置

在弹出窗口登录

如果上传出错，出现unable to access“”的情况

取消代理

大功告成

登录 用户名.github.io 查看即可
